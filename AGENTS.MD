## Project Summary
A premium portfolio website for a professional video editor, featuring a sleek black theme, smooth animations, and a focus on high-quality work and client testimonials. The site highlights expertise in DaVinci Resolve, Premiere Pro, and Canva.

## Tech Stack
- Framework: Next.js 15 (App Router)
- Styling: Tailwind CSS 4
- Animations: Framer Motion
- Icons: Lucide React, React Icons
- Components: Radix UI, Shadcn/ui (implied)

## Architecture
- `src/app`: Routes and page layouts
- `src/components`: Reusable UI components
- `src/lib`: Utility functions and shared logic

## User Preferences
- Black themed aesthetic
- High-quality content writing
- Subtle, non-laggy animations
- Mobile-first responsiveness
- Tool icons: DaVinci Resolve, Premiere Pro, Canva (animated)

## Project Guidelines
- Use 'use client' for interactive/animated components
- Follow clean, modern design principles
- Optimize video/image loading for performance
- No comments unless requested

## Common Patterns
- Framer Motion for scroll reveals and hover effects
- Responsive grid layouts for portfolio work
- Flexible flexbox/grid for mobile layouts
